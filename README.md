# Applied AI: NLP, Self-Organizing Maps & Reservoir Computing — D7041E

Lab reports from **Applied Artificial Intelligence (D7041E)** at Luleå University of Technology (2024–2025).

This course explored different paradigms of AI and machine learning through three labs:

- **Lab 1: NLP & Information Retrieval** — Trained Word2Vec embeddings with Gensim and built SVM text classifiers for document categorization.
- **Lab 2: High-Dimensional Representations, SOMs & Backpropagation** — Classified 21 European languages using hyperdimensional representations, implemented Self-Organizing Maps on MNIST, and trained MLPs with sigmoid vs. ReLU activations.
- **Lab 3: Reservoir Computing** — Built an Echo State Network (ESN) for Mackey-Glass chaotic time-series prediction using fixed-weight reservoirs with linear readout training.

**Technologies**: Python, Gensim, scikit-learn, NumPy, Matplotlib, SOM, ESN
